網頁端
md5
將資料做 MD5 加密
maam_jslib_run_script
執行 JavaScript 格式的字串內容
maam_jslib_account_auth
執行登入, 登出, 檢查等動作

系統端
maam_md5_hash
將資料做 MD5 加密
maam_verify_account
帳號驗證函式

用戶端程式存取內部資料
maam_lulib_init
初始化, 開啟存取內部資料
maam_lulib_exit
使用完畢, 釋放使用權
MAAM_LULIB_HEAD_SESSION
巨集, 找到串列上開頭的那筆 session 資料
MAAM_LULIB_TAIL_SESSION
巨集, 找到串列上結尾的那筆 session 資料
MAAM_LULIB_NEXT_SESSION
巨集, 找到串列上目前這筆 session 資料的後一筆
MAAM_LULIB_PREV_SESSION
巨集, 找到串列上目前這筆 session 資料的前一筆
MAAM_LULIB_IP_NTOH
巨集, 將網路格式的位址轉為字串格式
maam_lulib_kick_session
踢掉該使用者 (移除該筆 session 資料)
